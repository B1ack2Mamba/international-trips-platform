# v20.3 — отдельное окно карты связей

Что добавлено:

- новый экран `/dashboard/spaces/graph`
- текущий `/dashboard/spaces` оставлен как распределение разделов по зонам доступа
- в новом окне нет зон: только разделы и визуальные связи между ними
- разделы можно перетаскивать по свободному canvas
- связи создаются перетягиванием из одной карточки в другую
- состав узлов и связей хранится отдельно от распределения по зонам

Новые таблицы:

- `workspace_module_graph_nodes`
- `workspace_module_graph_links`
