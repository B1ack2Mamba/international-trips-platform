# v20 visual canvas

Цель этой версии — убрать внутренний builder экрана и оставить только визуальную топологию.

Что есть:
- свободное поле для зон доступа;
- drag зон по canvas;
- resize зон;
- drag разделов внутри зоны и между зонами;
- визуальные связи между разделами без бизнес-логики;
- библиотека системных и своих разделов слева;
- редактирование состава доступа по email прямо в шапке зоны.

Что хранится:
- `workspace_spaces.metadata.canvas` — координаты и размеры зоны;
- `workspace_space_modules.metadata.canvas` — координаты раздела внутри зоны;
- `workspace_visual_links` — визуальные связи между карточками.
