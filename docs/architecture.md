# Архитектурная идея

Платформа собрана как **единое ядро данных + несколько рабочих интерфейсов**.

## Мозг

- Dashboard руководителя
- аналитика
- KPI
- права доступа
- контроль качества
- outbox / reporting / visibility

## Двигатель

- лидогенерация
- маршрутизация лидов
- продажи
- скрипты
- follow-up
- partner attribution

## Коробка передач

- сделки
- заявки
- договоры
- оплаты
- дедлайны
- message dispatcher

## Колёса

- программы
- выезды
- группы
- участники
- логистика
- trip ops checklist

## Тормоза и стабилизаторы

- RLS
- роли
- anti-poach lock
- audit trail
- обязательные поля
- SLA
- OTP для family portal

## Радар

- формы сайта
- Telegram
- партнёры
- школы
- контекст / таргет
- реферальные каналы
- webhook intake
